SCASFLIX Server
Movie Streaming Backend

A powerful, scalable movie streaming API built with Node.js, Express, and MongoDB. Perfect for learning full-stack development or as a foundation for production applications.

// Get trending movies
GET /api/movies/trending

// User authentication
POST /api/auth/signin

// Add to My List
POST /api/list/{profileSlot}
✨ Key Features

Powerful Backend Capabilities

Everything you need for a modern streaming platform

🔐

JWT Authentication

Secure user authentication with JSON Web Tokens. Password hashing with bcrypt and token-based session management for maximum security.

🎬

Movie CRUD Operations

Complete Create, Read, Update, Delete operations for movies and TV shows. Seamlessly integrated with TMDB API for rich content data.

🔗

RESTful API Design

Properly structured REST endpoints with consistent response formats. Full error handling and validation for robust production use.

☁️

Cloud Deployment Ready

Optimized for deployment on Heroku, Render, or any Node.js hosting platform. Environment variables and database connection pooling included.
